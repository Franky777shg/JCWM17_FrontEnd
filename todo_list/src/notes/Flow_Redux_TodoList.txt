1. App.js dipanggil, lalu dia merender component navigationbar dan todopages
2. yang terjadi di navigationbar, component nya akan dirender, kemudian akan diconnect ke redux
3. yang terjadi di todopages, componentnya akan dirender dan di connect ke redux
4. componentDidMount jalan, lalu menjalankan function fetchData
5. function fetchData mengambil data dari json server, kemudian data nya dikirim melalui action getData
6. action getData akan menghasilkan sebuah object yang berisi type dan payload, dimana nanti type nya akan disesuaikan dengan case yang ada di reducer dan payload akan berisi data dari hasil axios
7. object hasil getData akan diteruskan ke todoreducer
8. sesampainya di todoreducer, case akan dijalankan sesuai type yang diterima yaitu 'GET_DATA'
9. case untuk type 'GET_DATA' mengisi data activities dengan data hasil get dari axios yang dibawa oleh payload
10. ketika data di todoreducer berubah, maka render akan dijalankan lagi
11. sehingga data yang sekarang adalah hasil dari data redux yang sudah berubah
